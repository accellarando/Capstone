Development installation instructions

Instructions for building the Kinect software and dependent libraries on Linux. Follow https://tayyabnaseer.blogspot.com/2012/05/installing-skeltrack-on-ubuntu.html with these minor changes.
  1. Get libfreenect.
  2. Get GFreenect.
  3. In GFreenect, change all instances of libfreenect >= 0.0 in configure.ac to libfreenect >= 0. It's on line 79 for me.
  4. ./autogen.sh
  5. Rest of the instructions are the same for GFreenect.
  6. For Skeltrack, edit tests/Makefile.am once you run into issues: https://github.com/joaquimrocha/Skeltrack/pull/31/commits/d52ab79ac76d4690e925ba0f630b673cdc3aa5f6
  7. If the executable complains about missing shared libraries, follow the highlighted parts of the Blogspot guide.
  8. You may also need to edit the application's Makefile to point to the libraries - see the prototype repo for an example.
